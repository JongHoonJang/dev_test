# dev_test 게시판 만들기

### 기술스텍

frontend : vue3

backend : Django

DB : PostgreSql

### 알고리즘

계층형게시판 알고리즘

group_order : 같은 주제를 갖는 게시물의 고유번호. 부모글과 부모글로부터 파생된 모든 자식글은 같은 번호를 갖습니다.

order_id : 같은 그룹내 게시물의 순서

depth : 같은 그룹내 게시물의 순서
