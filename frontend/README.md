# frontend

### 1. 시작하기

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### 2. 컴포넌트

1. NavBarView 

    - 로그인

    - 회원가입

    - 회원탈퇴

2. MainView(BoardListView)

    - 게시글 리스트

3. ModalView(SignupView)

    - 회원가입

4. BoardDetailView

    - 게시글 상세페이지

    - 게시글 삭제

    - 답글 작성

    

5. BoardCreateView

    - 게시글(댓글) 작성

6. BoardEditView

    - 게시글 수정
